proyectogit_equipo

Repositorio para realizar la tarea de git en programacion front-end

Para esta tarea desarrollamos un reproductor de música con 6 canciones elegidas por los integrantes del equipo.
El proyecto fue desarrollado utilizando HTML, CSS y JavaScript, aplicando buenas prácticas de organización y
control de versiones con Git y GitHub.

Cada integrante trabajó el proyecto desde Visual Studio Code, utilizando ramas para dividir el trabajo y
posteriormente unirlo en la rama principal.
